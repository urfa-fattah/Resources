Accession guide

Filing a new resource

  1. Drop the file into the /resources folder that sits next to index.html.
  2. Name it course‑first, e.g. CSE-1101-lecture-3-boolean-algebra.pdf — the part before the first dash or space is what gets matched to a course.
  3. List the filename in resources/manifest.json — it's a plain array of strings: 
    [
  "CSE-1101-lecture-3-boolean-algebra.pdf",
  "CSE-1102-lab-sheet-2.pdf",
  "MATH-1101-past-paper-2024.pdf"
]
  4. Click "Refresh library" (or reload the page) — the file appears filed under its course automatically.

Unrecognised prefixes land in Unfiled instead of being dropped — nothing is ever hidden.

Opening index.html by double‑clicking it won't work — browsers block local file reads that way. Serve the folder with python -m http.server (or any static host) and open it over http:// instead.
